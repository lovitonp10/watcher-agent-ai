Commandes Principales
=====================

Installation
------------
./setup.sh
source venv/bin/activate

Configuration
-------------
cp .env.example .env
nano .env  # Ajouter vos clés API

Tests
-----
python test_llm.py           # Tester le LLM
python main.py test-email    # Tester l'email
python test_e2e.py          # Test complet

Utilisation
-----------
python main.py update        # Mettre à jour la base
python main.py chat          # Chat interactif
python main.py search "..."  # Recherche rapide
python main.py digest        # Envoyer email digest
python main.py stats         # Statistiques
python main.py info          # Configuration

Automatisation
--------------
crontab -e
45 8 * * * /chemin/vers/watcher/schedule_digest.sh


Documentation Hiérarchie
========================

1. Débutant
   → README.md (5 min lecture)
   → QUICKSTART.md (10 min setup)

2. Personnalisation
   → SOURCES.md (ajouter des sources)
   → config.py (modifier la config)

3. Avancé
   → STRUCTURE.md (comprendre le code)
   → docs/EMAIL_SETUP.md (email avancé)
   → docs/PROVIDERS.md (changer de LLM)

4. Développeur
   → src/ (code source commenté)
   → test_*.py (exemples d'utilisation)


Flux de Travail Recommandé
===========================

Première Utilisation
--------------------
1. Lire README.md
2. Suivre QUICKSTART.md
3. Configurer .env
4. Tester : python test_llm.py
5. Lancer : python main.py update
6. Utiliser : python main.py chat

Usage Quotidien
---------------
1. Email automatique chaque matin (cron)
2. Questions ponctuelles : python main.py chat
3. Mise à jour manuelle si besoin : python main.py update

Personnalisation
----------------
1. Lire SOURCES.md
2. Modifier config.py (ajouter blogs/catégories)
3. Ajuster .env (keywords, days_to_fetch)
4. Tester : python test_ingestion.py
